# STEP1：デバイスIDを確認し、カイさんにDMで送る

## デバイスIDとはどこに？（macの場合）

WIFI -> 詳細 -> ハードウェア -> MACアドレス

この状態で、
00:0a:00:aa:a0:00 のようなアドレスと⾃⾝のメールアドレスをカイさんにDMで送信してください。

フォーマットは

デバイスID：00:0a:00:aa:a0:00
メールアドレス：aaa@2wins.ai

でよろしくお願いします。

## デバイスIDとはどこに？（Windowsの場合）

フォーマットは

デバイスID：00:0a:00:aa:a0:00

# STEP3：メールでデバイス証明書発⾏の通知を確認

ここから先はメールに従って⼿順通りに⾏ってください。

【手順１】
下記URLより、「Cybertrust DeviceiD Importer」をインストールして下さい。

リンクを押してダウンロードすると以下のような画⾯になると思います。
DeviceID importer を Applications にドラッグしてください。

そのあとアプリを⽴ち上げてください。

⽴ち上げる時に 証明書識別子 が必要になると思うのですが、これは最初のメールに記載があります。

## ⚠ 以下重要⚠

### ちなみにこの証明書識別⼦を⼊⼒する欄ですが、メチャクチャUIが悪く、コピペしても何も表⽰されません😭
ペースト後バックスペースを⼀度押すと⼊⼒された状態になりますので、そのあとに 証明書登録 をクリックしてください。
