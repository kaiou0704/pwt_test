# 統合仕様書：京都春の観光モデルコース

## 1. ユースケース概要
本ユースケースは、京都の歴史的建造物、自然、および花街の情緒を2日間で巡る観光プランの実行を目的とする。

## 2. 業務フロー

### 1日目：東山の歴史と花街の情緒
*   **10:00 京都駅到着**
    *   コインロッカーへ荷物を預け、観光を開始する。
*   **11:30 清水寺周辺**
    *   清水の舞台を参拝。
    *   産寧坂および二寧坂を散策する。
*   **15:00 祇園エリア**
    *   石畳の街並みを散策。
    *   白川沿いの桜を鑑賞し、甘味処で休憩をとる。
*   **18:00 ホテル到着**
    *   アゴーラ 京都烏丸へチェックイン。
    *   近隣にて京料理の夕食をとる。

### 2日目：嵐山の自然美
*   **09:00 ホテル出発**
    *   朝食後、阪急電車を利用して嵐山へ移動する。
*   **10:00 嵐山・渡月橋エリア**
    *   渡月橋からの景観を楽しむ。
    *   竹林の小径にて森林浴を行う。
*   **13:00 天龍寺周辺**
    *   世界遺産の庭園と桜を鑑賞。
    *   門前町にてランチおよび土産物の購入を行う。
*   **16:00 京都駅**
    *   土産物を購入し、旅を締めくくる。

## 3. 機能要件
*   **移動手段**: 2日目の嵐山への移動には阪急電車を使用すること。
*   **宿泊施設**: 宿泊先は「アゴーラ 京都烏丸」を指定する。
*   **観光対象**: 
    *   歴史的建造物（清水寺、天龍寺）
    *   自然景観（白川沿いの桜、渡月橋、竹林の小径、天龍寺庭園）
    *   街並み（産寧坂、二寧坂、祇園の石畳）
