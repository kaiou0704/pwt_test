# 統合仕様書：デバイス証明書の発行とHENNGE ONEログイン

## 1. 業務フロー概要
本プロセスは、デバイスIDの確認からデバイス証明書のインストール、およびHENNGE ONEへの初回ログインまでの一連の手順を定義する。

## 2. 機能要件・手順

### 2.1 デバイスIDの確認と共有
*   ユーザーは自身のデバイスIDを確認する。
*   確認したデバイスIDをカイ（担当者）へDMにて送付する。

### 2.2 デバイス証明書のインストール
*   ユーザーは、メールで通知されるデバイス証明書発行の案内を確認する。
*   指定されたURLより「Cybertrust DeviceiD Importer」をダウンロードし、アプリケーションフォルダへインストールする。
*   アプリケーションを起動し、初回メールに記載されている「証明書識別子」を入力して設定を完了する。

### 2.3 HENNGE ONEへのログインとパスワード設定
*   HENNGE Oneのログインページへアクセスする。
*   「証明書ログイン」を選択し、インストールした証明書を指定してログインを行う。
*   ログイン画面にて「パスワードをお忘れですか？」を選択し、パスワードの再設定を行う。
*   設定したパスワードを使用して、HENNGE Oneへログインする。ssss
